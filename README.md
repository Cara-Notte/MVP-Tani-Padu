# Tani Padu

MVP aplikasi web berbahasa Indonesia untuk pasar pangan Rajeg, dashboard produsen, dan monitoring pangan pemerintah.

## Menjalankan Proyek Secara Lokal
```bash
npm install
npm run dev
```

## Akun Demo

- Konsumen: `konsumen@tanipadu.id` / `demo123`
- Produsen: `produsen@tanipadu.id` / `demo123`
- Pemerintah: `pemerintah@tanipadu.id` / `demo123`

Registrasi akun baru selalu membuat akun dengan status Konsumen. Upgrade ke Produsen atau Pemerintah dilakukan dari halaman Profil.
